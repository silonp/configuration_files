## Tmux
Use `bin/tmux_start` script to start or attach to the session.
 - Linux `ln -s ~/configuration_files/tmux/tmux.conf ~/.tmux.conf`
 - Cygwin/MSYS2 `cmd //c 'mklink %HOME%/.tmux.conf %HOME%/configuration_files/tmux/tmux.conf'`
